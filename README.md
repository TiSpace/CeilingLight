# CeilingLight
Ceiling Light with Webserver, controlling 12V LED Sports, RGB Sting as well as WS2812B String
## Hardware
* NodeMCU 0.9
* (Power) n-MOS FET
* Strip RGB Light (common anode)
* Strip WS2812B
* couple of LED beams
